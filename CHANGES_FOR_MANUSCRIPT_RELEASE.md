- Removed the obsolete `[0.09, 0.10]` alpha-lower selection band from the documented main-design selection procedure.
- Retained the manuscript selection rule: maximize average working-alternative PRN-any, then minimum PRN-any, then the feasible maximum global type I error.
- Changed all scripts to use repository-relative `results/` and `tables/` directories through `ENRICHMENT_BOP2_ROOT`.
- Added generation and manuscript mapping for the three detailed binary enrichment-path tables.
- Isolated quick-check outputs from full-analysis outputs.
- Updated table-generator wording to match the current manuscript definitions, including cumulative biomarker-positive analyses and rounding of displayed 0.0 values.
